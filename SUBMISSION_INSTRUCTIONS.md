## How to Submit

### Option 1: Google Docs (Recommended)

1. **Open Google Docs** (docs.google.com)
2. **Create a new document**
3. **Copy the entire content** from `Solana_Socials_101_Playbook.md`
4. **Paste into Google Docs**
5. **Format the document:**
   - Add headings using Google Docs heading styles
   - Format code blocks (use "Courier New" font for code)
   - Add any visuals/screenshots you want
   - Ensure good spacing and readability
6. **Set sharing permissions:**
   - Click "Share" button (top right)
   - Change to "Anyone with the link can view"
   - Copy the shareable link
7. **Submit the link** to the hackathon

### Option 2: Notion
